# Tribuna: KPL Football Ticket Booking Platform

## Description
A full-stack web application that simulates a football ticket booking system for the Kazakhstan Premier League. Users can browse matches, select seats in a stadium layout, and simulate ticket purchases.

## Features
- View upcoming matches  
- Select match and stadium seats  
- Different seat sections (VIP, Regular)  
- Simulated ticket booking  
- Seat availability tracking  

## Tech Stack
- **Frontend:** Angular, TypeScript  
- **Backend:** Django  
- **Database:** SQLite
